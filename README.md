# 📊 Enterprise Business Intelligence Platform

An end-to-end Business Intelligence project built using **SQL Server, Python, Microsoft Excel, Tableau, and Power BI** to analyze sales, customers, products, and financial performance using the Microsoft AdventureWorks dataset.

---

# 📌 Project Overview

This project demonstrates the complete Business Intelligence workflow—from data preparation and SQL analysis to Python-based data cleaning and interactive dashboard development.

The project provides business insights into sales performance, customer behavior, product performance, profitability, and regional trends through multiple visualization tools.

---

# 🎯 Business Objectives

The project answers key business questions such as:

- What is the total revenue and profit?
- Which products generate the highest sales?
- Which product categories are the most profitable?
- Which regions contribute the most revenue?
- Who are the top customers?
- How do sales change over time?
- What is the average order value?
- Which customer segments generate the highest revenue?
